# TINY-RPC
TINY-RPC  
基于Netty的小型RPC项目  
目前为version_1.2版本  
|功能列表|  
|version1.0 : 仅仅为可运行的版本  
|version1.2 : 为TINY-RPC 添加 SpringBoot Stater  
|version1.3 : 添加Zookeeper，提供服务发现和注册  
|version1.4 : 添加负载均衡策略  
How to Run:  
在测试仓库(https://github.com/raylrnd/tiny-rpc-sample.git)中分别运行'ServerDemo'和'ClientDemo'
